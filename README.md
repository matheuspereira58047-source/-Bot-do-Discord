# 🤖 Bot do Discord

Bot de moderação feito em Python usando discord.py.

## 📌 Funcionalidades

* 🗑️ Limpar mensagens (`!clear`)
* 👢 Expulsar membros (`!kick`)
* 🔨 Banir membros (`!ban`)
* 🏓 Teste de ping (`!ping`)

## ⚙️ Tecnologias

* Python
* discord.py
* asyncio

## 🚀 Como usar

1. Instale as dependências:

```bash
pip install discord.py
```

2. Coloque seu token no código:

```python
bot.run("SEU_TOKEN")
```

3. Rode o bot:

```bash
python bot.py
```

## ⚠️ Aviso

Não compartilhe seu token do bot.
